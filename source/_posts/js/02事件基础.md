### 1.事件基础： 

简单理解为：触发--相应机制



### 2.事件组成：

 事件源 事件类型 事件处理程序--事件三要素

 ~~~js
   // <botton id="btn">按钮</button>
 
 	//1.事件源  事件被触发的对象
 	var btn=document.getElementById('btn');
 	//2.事件类型 如何触发           事件：鼠标点击onclick
 	//3.事件处理程序  通过一个函数赋值的方式  完成
 	btn.onclick=function(){   
 	console.log('点击按钮');
   }
 ~~~



### 3.执行事件的步骤：

   + 获取事件源  

   + 注册事件（绑定事件）  

   + 添加事件处理程序（采用函数赋值形式）

     

### 4.常见鼠标事件：	
   + onclick	         点击左键
   + onmouseover   鼠标经过
   + onmouseout     鼠标离开	
   + onfocus	         获得鼠标焦点
   + onblur	         失去鼠标焦点
   + onmousemove 鼠标移动
   + onmouseup      鼠标弹起
   + onmousedown 鼠标按下
