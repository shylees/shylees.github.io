判断是否到达页面底部

~~~js
$(window).scroll(function () {  
	if ($('html, body').scrollTop() + window.innerHeight >= Math.floor($('html').outerHeight(true))) {
    if (PART == 1) {
      loadingNextPART1();
    } else if (PART == 2) {
      loadingNextPART2();
    }
  }
}）
~~~
